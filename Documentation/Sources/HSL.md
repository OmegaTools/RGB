
# HSL

*Convert HSL colors to RGB.*

```JavaScript
import { fromHSL } from 'RGB'
```

<br>

## Array

*Pass the channels as an array.*

```JavaScript
const rgb = fromHSL([ H , S , L ]);
```

<br>

## Alpha

*Optionally include an Alpha channel.*

```JavaScript
const rgba = fromHSL([ H , S , L , A ]);
```
